# Ubuntu Command Cheatsheet

A quick reference guide for essential Ubuntu commands.

## Filesystem Commands

* **List Directory Contents:**
    `ls -l` 
    Shows all files and directories in long format (permissions, owner, size, date).

* **Change Directory:**
    `cd /path/to/directory`
    Navigates to the specified folder.
